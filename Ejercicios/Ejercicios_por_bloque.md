# 2. Ejercicios detallados por bloque

Aquí tienes **ejercicios ampliados**, listos para entregar como dossier.

---

## **Bloque 1 – Insertar y modificar elementos**

- Crear una tabla de inventario con:  
  - Autoformato  
  - Imagen por fila  
  - Texto girado  
  - Tabla anidada para observaciones  
- Crear un folleto turístico de Tineo con:  
  - Dos columnas  
  - Imágenes ajustadas  
  - Autoformas con texto  
- Crear un gráfico con datos de producción agrícola.  
- Crear un organigrama de una asociación local.  
- Crear un diagrama de proceso (por ejemplo, elaboración de queso).

---

## **Bloque 2 – Trabajo avanzado con documentos**

- Documento de 5 páginas con:  
  - Notas al pie  
  - Marcadores  
  - Referencias cruzadas  
  - Tabla de contenido  
- Documento maestro con capítulos:  
  - Historia del concejo  
  - Gastronomía  
  - Fiestas  
- Formulario de inscripción a un taller.  
- Macro que:  
  - Inserte un encabezado  
  - Aplique estilo  
  - Inserte fecha automática  
- Comparar dos versiones de un documento.

---

## **Bloque 3 – Gestión avanzada**

- Crear carta de invitación + origen de datos.  
- Crear etiquetas para envío postal.  
- Incrustar hoja de cálculo con gastos y calcular totales.  
- Crear documento con hipervínculos internos y externos.  
- Guardar como HTML.
